========================
База знаний кодов ошибок
========================

.. toctree::

Этот документ описывает ошибки, которые могут возникнуть в процессе взаимодействия с платформой. Каждый код ошибки имеет общий тип ошибки, и эти коды отличаются в зависимости от проблемы.
